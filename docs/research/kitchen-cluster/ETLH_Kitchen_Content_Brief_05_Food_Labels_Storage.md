# ETLH kitchen content brief 05 — Food labels and storage

| Field | Brief |
| --- | --- |
| Working title | Food Labels and Storage in a Small Apartment: What to Keep Visible |
| Route | `/blog/food-labels-storage-small-apartment` |
| Reader problem | Limited storage, shared shelves, loose containers, or optional transfers can separate food from useful package information. |
| Independent purpose | Preserve product and package information; not pantry organization, a decanting tutorial, a container-buying guide, allergy advice, or a food-safety guarantee. |
| Intent | Informational: understand which product details may be worth retaining and when leaving the original package is simplest. |
| Evidence | FDA date-label guidance; FDA allergen-label guidance; FDA recall-identification guidance; FoodKeeper only for food-specific storage details; package and manufacturer instructions for product-specific handling. |
| Allowed claims | Original packages may contain product identity, ingredients, allergens, dates, lot/batch details, directions, storage instructions, and manufacturer contact information; a handwritten label cannot recreate every field. |
| Prohibited claims | Illness/allergy/recall prevention, waste or savings outcomes, shelf-life extension, universal container suitability, or safety conclusions from a date alone. |
| Safety/accessibility | Infant-formula exception; ingredient changes; recalls; package integrity; child resistance; pets; allergies; vision, language, literacy, disability, shared-household, and rental constraints. |
| Outline | Information loss; decide whether transfer is necessary; information worth keeping visible; limits of secondary labels; when original packaging should stay; dates/allergens/infant formula/recalls; no-purchase check; sources and limits. |
| Internal links | Pantry organization for physical storage; food-scrap method for disposal decisions; kitchen habits for broader food-use planning. |
| Visual | Illustrative shelf/cabinet with intact unbranded packages and one labeled secondary container; no readable brands/dates or safety implication. |
| Alt text | Illustrative small apartment pantry shelf with intact food packages and one labeled secondary container. |
| Channel | Pinterest: information to retain before transferring food. Instagram: original package versus secondary label. No external publication. |
| Stop conditions | Pause for unsourced storage times, allergy/recall interpretation, incomplete secondary labels, or unsupported container advice. |
| Status | Approved for autonomous publication implementation by owner; article must still pass repository, CI, accessibility, and production gates. |
