# Small-Apartment Eco Step Starter Sheet v1

Production candidate for owner review. This package contains:

- `small-apartment-eco-step-starter-sheet-v1.pdf` - two-page US Letter printable worksheet.
- `small-apartment-eco-step-starter-sheet-v1.html` - responsive, keyboard-accessible HTML companion.
- `generate_eco_step_sheet.py` - reproducible PDF source.

## Boundaries

- No savings, environmental outcome, stress reduction, safety, renter approval, or habit-formation claim is made.
- The PDF is printable but is not represented as a fully tagged or fillable accessible PDF.
- The HTML companion provides labeled fields, logical reading order, focus visibility, and print support.
- No public CTA or email delivery behavior should change until owner review and implementation testing are complete.

## Review checklist

- Owner approves title, wording, examples, visual treatment, and direct-download model.
- Verify grayscale and common home-printer output.
- Verify 320 px and 390 px mobile layouts for the HTML version.
- Confirm final canonical URL and corrections/contact destination.
- Confirm repository location and deployment behavior.
